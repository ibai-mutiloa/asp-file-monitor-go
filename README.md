# ASP File Monitor in Go

## Project Overview
This project is a Go-based agent designed to monitor changes in ASP files on a Windows server and automatically commit them to a Git repository. It is intended to facilitate version control and traceability for legacy ASP applications.

## Features
- Monitors one or more directories for changes in `.asp` files.
- Automatically stages and commits changes to a configured Git repository.
- Configurable commit messages and intervals.
- Designed to run as a lightweight background service on Windows.
